# Lab 1.1 : Getting Started with lms-page

Welcome to the Interactive Lab Platform. In this lab, you will learn how to navigate the system and use the integrated terminal.

## Objectives
- Familiarize yourself with the interface.
- Run your first commands in the terminal.
- Understand how to submit your work.

## Task 1: Check System Information
Run the following command in the terminal to see your current user and hostname:

```bash
whoami && hostname
```

## Task 2: Explore Directories
List the files in your home directory:

```bash
ls -la
```

## Conclusion
Once you have explored the environment, click the **Next** button to proceed.
